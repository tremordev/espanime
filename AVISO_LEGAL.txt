NATURALEZA TÉCNICA Y OBJETO
Este software ("Espanime") es una herramienta que actúa como interfaz de presentación entre Kodi y páginas web de contenido audiovisual de anime. El addon realiza acceso a sitios web públicos de terceros, cargando la información expuesta públicamente (títulos, miniaturas, enlaces a reproductores) y presentándola en la interfaz de Kodi.
No existe una base de datos interna de enlaces ni un repositorio precargado de contenido, cada resultado se obtiene dinámicamente al ejecutar la consulta del usuario, actuando de forma similar a un navegador web estándar.
El contenido accesible a través de este addon también es accesible mediante un navegador web convencional visitando directamente los sitios web de origen.

INTEGRACIÓN CON SOFTWARE EXTERNO
Este software ofrece funcionalidades de interoperabilidad con otros addons y módulos de Kodi instalados por el usuario. Estas integraciones:
 a) Son mecanismos de delegación que redirigen la resolución y reproducción de enlaces al software externo correspondiente.
 b) No implican que este proyecto desarrolle, mantenga, controle o sea responsable de la funcionalidad de dichos addons externos.

 d) La resolución de enlaces de vídeo se delega en ResolveURL (módulo externo de Kodi que el usuario instala voluntariamente). ResolveURL es quien gestiona la obtención de la URL directa de reproducción desde los servidores de vídeo. La descarga de contenido se delega en youtube-dl (script.module.youtube.dl), igualmente externo y de instalación voluntaria por el usuario. Espanime no implementa lógica propia de resolución de servidores de vídeo ni de descarga de streams.

SERVICIOS DE TERCEROS CONSULTADOS AUTOMÁTICAMENTE
Además de los sitios web de anime scrapeados, este software realiza conexiones automáticas a los siguientes servicios de terceros para funcionalidades auxiliares:
 a) AniList (graphql.anilist.co): API GraphQL pública consultada para obtener metadatos enriquecidos de los animes (sinopsis, puntuación, géneros, póster HD, fanart). Las consultas incluyen el título del anime buscado. AniList puede registrar la dirección IP del usuario.
 b) AniSkip (api.aniskip.com): API pública consultada para obtener marcas temporales de openings y endings (funcionalidad de auto-skip). Las consultas incluyen el identificador del anime y el número de episodio que el usuario está reproduciendo. AniSkip puede registrar la dirección IP del usuario.
Ambas funcionalidades pueden desactivarse en los ajustes del addon. El usuario puede consultar las políticas de privacidad de estos servicios en sus respectivos sitios web.
